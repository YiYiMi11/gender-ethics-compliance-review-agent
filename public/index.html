01

什么是性别伦理合规审查

一套面向公司官方账号的内容预审智能系统。它不是辩论调解员或社会议题评论员，而是一道前置的内容安全防线。

性别伦理

识别并拦截可能引发性别对立的措辞，消除无意识偏见，确保公司言论的包容性与公正性。

公序良俗

同步审查内容是否违反社会公序良俗，确保公司言论在法律底线和道德风尚上均无瑕疵。

02

六大审查原则

所有审查判定均基于以下六项基本原则，它们是知识库的上位准则。

01 不强化性别刻板印象 +

避免暗示"全体男性都……""女性天生就……"等对全性别进行特征归因的表述。不将特定性格、能力、角色与某一性别绑定。

02 不对立归因 +

不把一个群体的负面行为归因于整个性别，不构建"受害者 vs 压迫者"的二元框架。避免将社会问题简化为性别对立。

03 区分"事实陈述"与"价值评判" +

允许有可靠数据支撑的客观事实陈述，禁止带有价值判断的性别评价性表述。事实需要注明来源，价值评判应替换为中性描述。

04 公司言论的"安全边界" +

公司官方发言只讨论产品与服务、社会责任与ESG行动、通用价值观，不站队、不挑起性别政治争论。

05 具体化 → 中性化 → 均衡化 +

涉及性别话题时，按三步处理：具体化表述（避免泛化归因）、使用中性词汇（去除性别标签）、均衡对称举例（确保多角度呈现）。

06 遵守社会公序良俗 +

所有对外内容必须遵守公共秩序和善良风俗，公序良俗审查优先于性别伦理审查。任何违反法律法规和社会公德的内容均须立即拦截。

三大扫描层级

从高到低的三级扫描优先级，P0 确保红线零遗漏，P1 提供改写空间，P2 捕捉无意识偏见。

P0

绝对禁止类

→ 红色拦截

  • 性别与智力/能力/诚信直接挂钩
  • 以公司身份站队极端男权/女权派别
  • 性别与其他身份维度的交叉歧视

→ 橙色高可疑

  • 隐含性别角色标准的句式
  • 单一性别强调

→ 黄色需复审
